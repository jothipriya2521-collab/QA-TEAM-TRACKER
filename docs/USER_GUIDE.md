## How to Start the Tool

### Option 1: Local (Jothipriya starts server)
1. Go to folder: Norton/QA-Team-Tracker
2. Double-click START_TRACKER.bat
3. Share the IP link with your team
4. Keep the window open

### Option 2: Cloud (Vercel/Render)
- Access via deployed URL
- No local setup needed

## Daily Workflow

1. Login to tracker
2. Select Project, enter Hours, write Task Details
3. Click Save
4. Add multiple entries for different projects
5. Submit before 5:40 PM deadline

---

## Features

- Multi-project tracking per day
- Jira integration (worklog + comments)
- Email report generation (Lead only)
- CSV export (Lead/Managers)
- Password change via Profile

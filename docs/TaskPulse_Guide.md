## 1. What is TaskPulse?

TaskPulse is a web-based daily task tracking tool for the QA team. It allows team members to log their daily work, track hours across multiple projects, and managers to monitor team progress in real-time.

**Key Benefits:**
- Track who is working on what project and for how many hours
- Multi-project support (work on Norton 4h + RFC Migration 4h in same day)
- Custom date range filtering (view entries for any period)
- Edit profile and change password
- Managers can view real-time team progress without asking
- One-click email report generation for daily status
- CSV export for records
